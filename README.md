# ChatGPT Conversation Exporter

ChatGPTのプロジェクトページから、会話データをプロジェクト名・チャットタイトル・URL・エクスポート日時付きで、CSV/JSON形式でエクスポートする拡張機能です。

## 機能

- 📊 **テーブル表示** - 見やすい表形式で表示
- - 📋 **CSV出力** - スプレッドシート互換
  - - 📄 **JSON出力** - API/プログラム処理用
    - - 📥 **ダウンロード** - ファイルとして保存可能
      - - 📌 **プロジェクト名自動取得** - 所属プロジェクトを自動判定
        - - ⏰ **タイムスタンプ** - エクスポート日時を記録
         
          - ## インストール方法
         
          - 1. ZIPをダウンロード
            2. 2. 右クリック > 「ここに展開」
               3. 3. `chrome://extensions/` を開く
                  4. 4. 「デベロッパー モード」をON
                     5. 5. 「パッケージ化されていない拡張機能を読み込む」をクリック
                        6. 6. 解凍フォルダを選択
                          
                           7. ## 使い方
                          
                           8. 1. ChatGPTのプロジェクトページを開く
                              2. 2. 拡張機能アイコンをクリック
                                 3. 3. 「📊 データ抽出」ボタンをクリック
                                    4. 4. テーブル/CSV/JSONから選択
                                       5. 5. 「コピー」または「ダウンロード」
                                         
                                          6. ## 出力形式
                                         
                                          7. ### CSV
                                          8. ```
                                             project,title,url,exported_at
                                             "仕事関係","MacbookNeo提案","https://chatgpt.com/g/g-p-...","2026-04-01T12:00:00.000Z"
                                             ```

                                             ### JSON
                                             ```json
                                             [
                                               {
                                                 "project": "仕事関係",
                                                 "title": "MacbookNeo提案",
                                                 "url": "https://chatgpt.com/g/g-p-...",
                                                 "exported_at": "2026-04-01T12:00:00.000Z"
                                               }
                                             ]
                                             ```
